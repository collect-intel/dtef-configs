# Contributing to DTEF Blueprints

Blueprints in this repository are **generated by the dtef-app CLI**, not manually authored. To contribute new evaluation content:

1. Add or improve survey data adapters in [dtef-app](https://github.com/collect-intel/dtef-app)
2. Generate blueprints using the CLI tools
3. Submit a PR with the generated output
